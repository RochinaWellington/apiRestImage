# Project API -IMAGE
Api-rest upload image  to cloudinary. Connect mongoose DB storage product.
# environment variable
create file **.env**, add environment variable
```env
/** connect mongoose DB */
MONGODB_URI

/** connect cloudinary*/
CLOUDINARY_CLOUD_NAME
CLOUDINARY_API_KEY
CLOUDINARY_API_SECRET
```
# init project
```sh
npm install
npm run dev
```
